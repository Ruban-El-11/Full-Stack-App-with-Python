# Student Management System using Python Tkinter

## Project Overview

The Student Management System is a desktop-based application developed using Python and Tkinter. The application provides an easy and efficient way to manage student records through a graphical user interface (GUI). Users can add and view student information such as Name, Register Number, Degree, Department, Year, and CGPA.

## Features

* Add new student records
* Store student details
* Display records in a table format
* User-friendly graphical interface
* Success notification after adding records
* Simple and efficient data management

## Technologies Used

* Python 3
* Tkinter (GUI Library)
* ttk Treeview (Table Display)

## Required Fields

The application stores the following student information:

* Student Name
* Register Number
* Degree
* Department
* Year
* CGPA

## Installation

1. Install Python 3 from the official Python website.
2. Verify Python installation:

```bash
python --version
```

3. Download the project files.
4. Run the application:

```bash
python app.py
```

## How to Use

1. Launch the application.
2. Enter student details in the provided fields.
3. Click the "Add Student" button.
4. The student record will be displayed in the table.
5. Repeat the process to add multiple students.

## Output

The application opens a graphical window where users can enter student information and manage records efficiently through an interactive interface.
